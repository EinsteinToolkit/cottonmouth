# [Cottonmouth](https://github.com/einsteintoolkit/Cottonmouth)

<img
src="https://github.com/eschnett/CarpetX/blob/main/figures/carpetx.png"
width="200" />

**Cottonmouth** is a [Cactus](https://cactuscode.org/) thorn that implements the BSSNOK equations. It was designed for use with the CarpetX driver and was generated using [the Einstein Engine](https://github.com/max-morris/EinsteinEngine.git).

## Overview

Cottonmouth is ready for production. 

## Getting started

You might want to try take a look at the test parameter file (arrangements/Cottonmouth/CottonmouthBSSNOK/test/qc0.par) to see what it can do.
