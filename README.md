# [Cottonmouth](https://github.com/einsteintoolkit/Cottonmouth)

**Cottonmouth** is a [Cactus](https://cactuscode.org/) thorn that implements the BSSNOK equations. It was designed for use with the CarpetX driver and was generated using using [EmitCactus](https://github.com/mmor115/EmitCactus.git).

## Overview

Cottonmouth is ready for production. 

## Getting started

You might want to try take a look at the test parameter file (arrangements/Cottonmouth/CottonmouthBSSNOK/test/qc0.par) to see what it can do.
